# Retail Sales Analytics Dashboard

## Project Overview
This project analyzes 100,000+ retail sales records using SQL and Power BI to identify business insights.

## Tech Stack
- Java
- MySQL
- SQL
- Power BI
- Excel

## Features
- Revenue Analysis
- Profit Analysis
- Category-wise Sales
- City-wise Sales
- Payment Method Distribution
- Top Selling Products

## Dataset
- Customers: 10,000
- Orders: 50,000
- Order Details: 100,000
- Products: 50
- Stores: 40

## Key Insights
- Revenue by Year
- Profit Analysis
- Category-wise Sales
- City-wise Sales
- Payment Method Distribution
- Top Selling Products
