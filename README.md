# 📊 Retail Sales Analytics Dashboard

## Project Overview

Built an end-to-end Retail Sales Analytics solution to analyze 100,000+ sales transactions and generate actionable business insights. The project involved data ingestion, transformation, analysis, and visualization using Java, MySQL, SQL, and Power BI.

The dashboard enables business users to monitor revenue, profit, sales performance, customer purchasing patterns, and product trends through interactive visualizations.

## Tech Stack

* Java
* MySQL
* SQL
* Power BI
* Microsoft Excel
* JDBC

## Business Problem

Retail businesses generate large volumes of sales data every day, making it difficult to track performance and identify growth opportunities. This project was developed to centralize sales data and provide meaningful insights through an interactive dashboard.

## Project Workflow

1. Imported retail sales data from CSV files into MySQL using Java and JDBC.
2. Performed data cleaning and validation to ensure data quality.
3. Created SQL queries, joins, aggregations, and views for analysis.
4. Built KPIs and visualizations in Power BI.
5. Generated business insights to support data-driven decision making.

## Dashboard Features

* Revenue Analysis
* Profit Analysis
* Sales Trend Analysis
* Category-wise Sales Performance
* City-wise Sales Distribution
* Payment Method Analysis
* Top Selling Products
* Interactive Filters and Drill-Down Reports

## Key Insights

* Identified the highest revenue-generating years and sales trends.
* Analyzed profit distribution across different product categories.
* Compared sales performance across multiple cities and stores.
* Evaluated customer payment preferences.
* Identified top-performing products contributing most to revenue.
* Enabled quick business performance tracking through KPI dashboards.
